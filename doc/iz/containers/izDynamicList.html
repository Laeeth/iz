public @trusted @nogc nothrow class izDynamicList(T) : izList!T

An izList implementation, slow to be iterated, fast to be reorganized. This is a standard doubly linked list, with GC-free heap allocations.

While using the array syntax for looping should be avoided, foreach() processes with some acceptable performances.
