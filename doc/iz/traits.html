public static bool isCompatible(I, S)() 
if(is(S == struct) && is(I == interface))

Checks if the methods of an interface can be found within a structure.

The first version, slower, is able to diagnose the incompatibility. The second version short, performs a one time check base on the method type.

Parameters

I

the interface used as mmodel.

S

the structure which has to be checked.

Return

true if I methods are found in S.
