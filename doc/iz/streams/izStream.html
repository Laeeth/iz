public interface izStream

Defines the members of stream implementation. 

public size_t read(
    izPtr aBuffer, 
    size_t aCount)

Reads aCount bytes from aBuffer. Returns the count of bytes effectively read. 

public size_t readVariable(T)(T* aValue)

Read T.sizeof bytes in aValue. Returns the count of bytes effectively read (either T.sizeof or 0). T must verify isConstantSize. A typed reader is generated for each type in izConstantSizeTypes and named readint, readchar, etc. 

public size_t write(
    izPtr aBuffer, 
    size_t aCount)

Writes aCount bytes to aBuffer. Returns the count of bytes effectively written.
