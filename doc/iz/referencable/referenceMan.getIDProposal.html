public static ulong getIDProposal(RT)(RT* aReference)

Proposes an unique ID for a particular reference. This is a convenience function which will not return the same values for each software cession. A better user solution is to use the hash of an identifier chain (e.g the hash of "wizard.lefthand.magicwand")

Parameters

RT

a referencable type. Optional, likely to be infered.

aReference

a pointer to a RT.

Returns

the unique ulong value used to identify the reference.
