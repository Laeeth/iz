public class izPropertyBinder(T)

This container maintains a list of property synchronized between themselves.

The reference to the properties are stored using the izPropDescriptor format. The izPropDescriptor name field can be omitted.

Parameters

T

the common type of the properties. 

public ptrdiff_t addBinding(
    ref izPropDescriptor!T aProp, 
    bool isSource = false)

Adds a property to the list. If the binder is not local then aProp should neither be a local descriptor, otherwise the descritpor reference will become invalid.

Parameters

aProp

an izPropDescriptor of type T.

isSource

optional boolean indicating if the descriptor is used as the master property. 

public izPropDescriptor!T* newBinding()

Adds a new property to the list. The life-time of the new descriptor is handled internally.

Returns

an new izPropDescriptor of type T. 

public void removeBinding(size_t anIndex)

Removes the aIndex-nth property from the list. The item is freed if it has been allocated by _newBinding_. _source_ might be invalidated if it matches the item.

Parameters

anIndex

the index of the descriptor to remove. 

public void change(T aValue)

Triggers the setter of each property. This method is usually called at the end of a setter method (in the _master_/_source_ setter).

Parameters

aValue

a value of type T to send to each slave of the list. 

public void updateFromSource()

Call _change()_ using the value of _source_.
