public template izPropertiesAnalyzer()

When mixed in a class, several analyzers can be used to automatically create some izPropertyDescriptors for the properties or field anotated with @SetGet, @Set and @Get. These analyzers can be called in every non-static methods, usually this(). 

private void*[] descriptors

Contains the list of izPropDesrcriptors created by the analyzers. getDescriptor() can be used to correctly cast an item. 

public final size_t descriptorCount()

Returns the count of descriptor the analyzers have created.

Returns a pointer to a descriptor according to its name.

Returns

protected final void* getUntypedDescriptor(string name)

Returns a pointer to a descriptor according to its name. Identical to the getDescriptor() except that the result type has not to be specified. 

protected final void analyzeAll()

Performs all the possible analysis. 

protected void analyzeFields()

Creates the properties descriptors for each field marked with @SetGet and whose identifier starts with one of the following prefix: underscore, f, F. The resulting property descriptors names don't include the prefix. 

protected void analyzeVirtualSetGet()

Creates the property descriptors for the setter/getter pairs annotated with @Set/@Get.To be detected the methods must be virtual and non final. In a class hierarchy, an overriden accessor replaces the ancestor's one.
