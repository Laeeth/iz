alias WantDescriptorEvent = void delegate(
    const(izSerNodeInfo*) nodeInfo, 
    out void* matchingDescriptor, 
    out bool stop)

Parameters

nodeInfo

the information the callee can use to determine the descriptor to return.

matchingDescriptor

the callee can set a pointer to the izPropertyDescriptor matching to the info.

stop

the callee can set this value to false in order to stop the restoration process.
