izEnumSet

An izEnumSet allows to create a bit field using the members of an enum. It's designed in a very similar way to the 'Set Of' types from the Pascal languages.

izEnumProcs

CallTable based on an enum. It can be compared to an associative array of type E[T]. Additionally an izEnumSet can be used to fire a burst of call.
